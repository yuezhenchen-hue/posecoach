# 智拍指南 更新日志

## v1.4 - 核心拍摄能力升级 (2026-04-02)

对标产品功能架构"第一层：核心拍摄能力"，全面补齐6大功能模块。

### 1. AI 场景识别 — 36种场景自动识别
- 从12种扩展至36种场景类型
- 新增：日出、森林、湖泊、瀑布、沙漠、星空、桥梁、霓虹街景、地铁、博物馆、图书馆、健身房、人像、合照、舞台、婚礼、美食、宠物、集市、节日、运动、水下、航拍、微距等
- 每种场景配套独立的创意拍摄技巧
- Vision 分类器标签映射覆盖200+关键词

### 2. 计算摄影引擎 — HDR / 夜景 / 多帧合成
- 夜景模式：自动检测低光环境，启用高质量优先级拍摄 + AI低光增强后处理
- HDR：场景联动自动开启
- 夜景处理中实时状态提示

### 3. 专业手动模式 — ISO / 快门 / 白平衡 / RAW
- 快门速度：1/1000s ~ 1s 共11档可选，支持手动/自动切换
- RAW 拍摄：检测设备RAW支持，一键开启RAW格式输出
- ISO：手动滑块 + 自动切换
- 白平衡/色温：2000K~10000K 手动调节

### 4. AI 超分辨率 — 低光细节增强与放大
- ImageEnhancer 模块：
  - `enhanceLowLight()`: 曝光提升 + 降噪 + 对比度 + 锐化四步流水线
  - `superResolution()`: Lanczos缩放 + UnsharpMask + 暗部细节恢复
  - `autoEnhance()`: 基于Core Image自动优化
- 拍照完成页新增"AI优化"和"超分辨率"一键按钮

### 5. 实时构图辅助 — 黄金分割 / 水平仪 / 引导线
- 电子水平仪：基于CoreMotion加速度计，实时显示倾斜角度
- 水平时绿色提示，倾斜时橙色警示
- 顶栏新增水平仪开关按钮

### 6. 电影级视频模式 — 防抖 / 慢动作
- 拍摄模式栏：自动 / 夜景 / 人像 / 专业 / 视频 / 慢动作
- 视频录制：AVCaptureMovieFileOutput，录制时红色脉冲指示+计时器
- 电影级防抖：preferredVideoStabilizationMode = .cinematic
- 慢动作：120fps 高帧率模式
- 视频自动保存到相册
